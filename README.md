HTML Resume Page
This project is a single page resume website built using only HTML and plain CSS as a part of the **Web Dev Cohort 2026** assignment.

Features
- Pure HTML & CSS
- Clean and readable structure
- Semantic HTML elements
- Sections for About, Skills, Experience, Projects, Education, and Contact

Tech Used
-HTML & CSS

How to Run 
1. Clone the repository
2. Open "index.html" in any browser

<br> You can simply visit "https://shiikharrr.github.io/Resume/"
